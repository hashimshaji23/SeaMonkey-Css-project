# 🌊 Sea Monkey Expeditions

## 📌 Project Overview

Sea Monkey Expeditions is a modern and responsive travel website designed to showcase wildlife and adventure experiences in Jordan. The website provides users with an engaging interface to explore different expedition options such as hiking, camping, sea safari, and wildlife tours.

This project focuses on delivering a visually appealing design using videos, images, and structured layouts to enhance user experience.

---

## 🚀 Features

- 🎥 Background video for immersive experience
- 📱 Fully responsive design (mobile & desktop)
- 🧭 Navigation bar with smooth section links
- 🏕️ Expedition categories (Hiking, Camping, Sea Safari, Vehicle Safari)
- 🖼️ Image gallery section
- 🗺️ Trip cards with details (location, duration, group size)
- 🎨 Clean and modern UI design

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **Custom Fonts** – For unique typography  
- **Media (Images & Video)** – For rich UI experience  

---

## ⚙️ How It Works

- The website uses **HTML** to structure sections like header, about, expeditions, and gallery.
- **CSS** is used to style the layout, fonts, and responsiveness.
- A **background video** is used in the header to create a dynamic visual effect.
- Users can navigate through sections like:
  - About
  - Expeditions
  - Gallery
